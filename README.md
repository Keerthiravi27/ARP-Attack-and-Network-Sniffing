## ARP Attacks:  
ARP spoofing: A hacker sends fake ARP packets that link an attacker's MAC address with an IP of a computer already on the LAN. 
Boot kali and Windows7 virtual machines.
In windows 7 give the command arp -a

## OUTPUT:
```
1)Open Ettercap from Applications.
2) An authentication/setup window will appear.
Select Unified Sniffing
Click OK
3) A list of network interfaces will appear.
Select eth0
Click OK
4) From the top menu:
Click Hosts
Select Scan for Hosts
5) After the scan is completed:
Again click Hosts
Select Hosts List
6) Two IP addresses will be displayed. Example:
192.168.1.1 → Router/Gateway IP address
192.168.1.5 → Another device connected to the same network
7) This process is used to identify devices connected within the same network.
```
